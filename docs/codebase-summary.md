## Mock Analysis

The `dashboard_mock.html` reveals the target design:
- **Dark theme** with glassmorphism panels (`#0A0A0A` background)
- **Purple primary** color (`#8B5CF6`)
- **Left sidebar** with navigation: Overview, Topics, Consumers, Schemas
- **Header bar** with cluster selector dropdown, search (Cmd+K), notifications, settings
- **Metric cards**: Cluster Status, Brokers (3/3), Topics (142), Partitions (1.2 TB)
- **Brokers table**: ID, Host/IP, Port, Partitions, Size, Status indicators
- **Typography**: Space Grotesk display, JetBrains Mono data, Material Symbols icons
- **Design language**: Rounded corners, subtle borders, hover transitions, pulsing status dots

## Key Features Implemented

### Topic Templates & Cloning ✅
- **Template Management**: Create, edit, delete, import/export topic configuration templates
- **Auto-Matching**: Apply templates automatically using glob patterns for topic names
- **Topic Cloning**: Clone existing topics with their configuration, partitions, and replication factors
- **Settings Integration**: Template management integrated into global settings overlay

### UI Components
- **Template List Table**: Display all templates with actions for edit/delete
- **Template Picker Dropdown**: Select templates during topic creation
- **Save Template Modal**: Save topic configurations as reusable templates
- **Template Settings Panel**: Configure template matching and management options
- **Clone Flow**: Seamless topic cloning with pre-populated configuration

### Backend Services
- **TemplateService**: Full CRUD operations with file persistence
- **Template Storage**: JSON-based storage in `~/.watermark/templates.json`
- **File Dialog Integration**: Native OS file dialogs for import/export operations
- **Comprehensive Testing**: 384-line test suite covering all template operations

### Dependencies & Integrations
- **Wails v2**: Native desktop app framework with Go backend
- **TanStack Router**: Client-side routing and navigation
- **React Query**: Data fetching and state management
- **TypeScript**: Strong typing throughout the codebase
- **shadcn/ui**: Component library with accessible design
- **Tailwind CSS**: Utility-first styling system
