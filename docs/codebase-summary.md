# Codebase Summary

## Overview

The Watermark project is currently in its **scaffolding phase**, initialized from the official Wails v2 React-TypeScript template. The codebase contains the boilerplate structure with a "Greet" example function, plus a UI mockup for the target dashboard design.

### Backend (Go) — Current State

```
main.go  → Wails bootstrap, window config, asset embedding
app.go   → App struct, startup lifecycle, Greet() binding
```

**Key patterns**:
- `//go:embed all:frontend/dist` — embeds compiled frontend into Go binary
- `wails.Run()` — configures window title, dimensions, background color, bound methods
- All public methods on `App` struct are auto-exposed to frontend JS

## Mock Analysis

The `dashboard_mock.html` reveals the target design:
- **Dark theme** with glassmorphism panels (`#0A0A0A` background)
- **Purple primary** color (`#8B5CF6`)
- **Left sidebar** with navigation: Overview, Topics, Consumers, Schemas
- **Header bar** with cluster selector dropdown, search (Cmd+K), notifications, settings
- **Metric cards**: Cluster Status, Brokers (3/3), Topics (142), Partitions (1.2 TB)
- **Brokers table**: ID, Host/IP, Port, Partitions, Size, Status indicators
- **Typography**: Space Grotesk display, JetBrains Mono data, Material Symbols icons
- **Design language**: Rounded corners, subtle borders, hover transitions, pulsing status dots
